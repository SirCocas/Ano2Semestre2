To install syntax highlight for ANTLR4 in vim do the following:

1. copy antlr4.vim to syntax directory:

$ mkdir --parents ~/.vim/syntax/
$ cp vim/antlr4.vim ~/.vim/syntax/

2. edit .vimrc file:

$ vi ~/.vimrc

, and add the following lines at the end:

syntax on
au BufRead,BufNewFile *.g4 set filetype=antlr4
