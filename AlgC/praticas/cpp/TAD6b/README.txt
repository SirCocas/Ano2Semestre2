CONTEÚDO:

O arquivo TAD6b.zip contém vários ficheiros novos e alguns modificados para
complementar e corrigir os fornecidos no arquivo TAD6.zip da semana passada.

Inclui:

* Novos módulos para implementar 3 algoritmos para os caminhos mais curtos.
* Função adicional no módulo GRAPH.
  --- CUIDADO: não substituir diretamente o vosso ficheiro Graph.c,
  para não perderem trabalho já feito
* Pequena correção no ficheiro IntegersQueue.c (asserção na criação).
* Nova Makefile que acrescenta as regras para compilar os novos módulos
  e o novo programa de teste example3.

SUGESTÃO:

Para integrar os ficheiros de TAD6 com TAD6b, sugere-se que proceda assim:

1) Desenpacote TAD6b.zip.  Isto produz a pasta TAD6b.
2) Execute `meld TAD6 TAD6b` para ver as diferenças entre essas pastas.
3) Selecione os ficheiros que só estão em TAD6 (a verde) e `Copy to right`.
4) Clique em `Graph.c` para ver as diferenças e transfira o seu código
   para a versão em TAD6b.
5) Pode analisar as diferenças introduzidas nos outros ficheiros modificados
   (assinalados a azul).
6) Passe a trabalhar em TAD6b.
